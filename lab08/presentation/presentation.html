Слайд 2

Введение в длинную арифметику

  • Что это? Методы работы с числами, не помещающимися в стандартные типы данных (>128 бит).
  • Применение: Криптография, компьютерная алгебра, точные научные расчёты.
  • Представление числа:
    • Основание системы счисления b (часто b = 2³², 2⁶⁴).
    • Число X = (x_{n-1} ... x₁ x₀)_b, где 0 ≤ x_i < b.
    • Знак хранится отдельно.

Слайд 3

Алгоритм сложения

Вход: Два n-разрядных числа u, v; основание b.

Выход: Сумма w = w₀ w₁ ... w_n (w₀ — перенос).

Шаги:

  1. j = n, k = 0 (перенос).
  2. w_j = (u_j + v_j + k) mod b; k = ⌊(u_j + v_j + k) / b⌋.
  3. j--. Если j > 0 → шаг 2.
  4. w₀ = k.

Слайд 4

Алгоритм вычитания

  • Вход: u ≥ v; n-разрядные; основание b.
  • Выход: Разность w = w₁ ... w_n.
  • Шаги:
    1. j = n, k = 0 («заём»).
    2. w_j = (u_j - v_j + k) mod b; k = ⌊(u_j - v_j + k) / b⌋.
    3. j--. Если j > 0 → шаг 2.

Слайд 5

Алгоритм умножения «столбиком»

Вход: u (n разр.), v (m разр.); b.

Выход: w = u * v (m+n разр.).

Основные шаги:

  • Инициализация w нулями.
  • Внешний цикл по j (разряды v).
  • Внутренний цикл по i (разряды u) с накоплением переноса k.
  • Запись результата в w.

Слайд 6

Алгоритм «быстрого столбика»

  1. Идея: Упрощение классического алгоритма за счёт группировки вычислений.
  2. Шаги:
    1. t = 0.
    2. Для каждого s от 0 до m+n-1:
      • Суммирование произведений разрядов u и v, дающих текущий разряд результата.
      • w_{m+n-1-s} = t mod b; t = ⌊t / b⌋.

Слайд 7

Алгоритм деления

Вход: Делимое u (n+1 разр.), делитель v (t+1 разр.), n ≥ t, v_t ≠ 0.

Выход: Частное q, остаток r.

Основные шаги:

  1. Нормализация и подбор цифр частного.
  2. Итеративное вычитание кратных делителя.
  3. Коррекция цифры частного и остатка при необходимости.
